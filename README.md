# ⭐ Leo's World — Leo's Wereld

A fun, free learning website made just for Leo!
English 🇬🇧 + Dutch 🇳🇱

## 🎮 Games included

| Game | English | Dutch |
|------|---------|-------|
| 🔢 Count | Count objects | Tellen |
| ➕ Add | Simple sums with visuals | Optellen |
| 🎨 Colours | Name the colour | Kleuren herkennen |
| 🧠 Memory | Find the pairs | Paren vinden |
| 🐘 Bigger | Which number is more? | Welke is groter? |
| 🌤️ Science | Nature & world questions | Natuur vragen |

## 🚀 How to put this on the internet (GitHub Pages)

1. Go to [github.com](https://github.com) and sign in
2. Click **New repository** (green button)
3. Name it: `leo-world`
4. Make it **Public**
5. Click **Create repository**
6. Upload the `index.html` file (drag and drop works!)
7. Go to **Settings → Pages**
8. Under **Source**, choose `main` branch → Save
9. Wait 1–2 minutes
10. Your site is live at: `https://YOUR-USERNAME.github.io/leo-world`

## ✏️ Want to add more questions?
Open `index.html` in any text editor.
Find the `SCIENCE` array to add more science questions.
Each question needs: `en` (English), `nl` (Dutch), `a` (answer), `opts` (4 choices).
